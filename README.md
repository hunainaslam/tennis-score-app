# Tennis Scoring App
This is a Tennis Scoring App.

## Prerequisites

- Node 8.11.2

## Recommendations

- Node manager to ensure node version is correct for this project
  - NVM works well: https://github.com/creationix/nvm

## Install the projects dependencies

```bash
npm install
```
## Run the app

```bash
node index.js
```
